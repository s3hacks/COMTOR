# COMTOR

# Requirements
Python
OpenCv
Flask
tweepy
flask-twitter-oembedder
Pygame

# Installation

Clone the repository
$ git clone https://github.com/s3hacks/COMTOR.git

Create a Virtual Environment (Optional) : https://docs.python.org/3/tutorial/venv.html

# Execution

$ cd COMTOR

$ python routes.py
